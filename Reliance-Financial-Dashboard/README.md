# Reliance Financial Dashboard

## Overview
This Power BI dashboard analyzes the financial performance of Reliance Telecom using KPI reporting and interactive data visualization.

## Tools Used
- Power BI
- Microsoft Excel
- DAX
- Power Query

## Dashboard Features
- Financial KPIs
- Revenue Analysis
- Expense Analysis
- Budget vs Revenue Comparison
- Collection Efficiency
- Risk Analysis
- Monthly Performance Trend

## Key Insights
- Compared revenue and operational expenses.
- Tracked monthly financial performance.
- Visualized collection efficiency and audit risk.
- Created interactive dashboards using slicers and charts.
